Перед началом работы:
1) Выполните скрипт CreateTable.sql. Он создаст необходимую базу данных с таблицами
2) В скрипте ImportTable.sql измените все пути к Excel-таблице на свой полный путь. Менять в строчках с OPENROWSET
3) Запустить скрипт

Все данные из Excel-таблицы будут переданы в базу данных
